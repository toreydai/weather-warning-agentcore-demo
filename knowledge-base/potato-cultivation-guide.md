# 内蒙古马铃薯栽培技术手册

## 一、播前准备

### 1.1 选地整地
- 选择土层深厚、排水良好的沙壤土或壤土，pH 5.5-7.0
- 前茬以禾谷类作物为宜，避免与茄科作物连作（至少间隔3年）
- 秋季深翻25-30厘米，春季旋耕整平，起垄高20-25厘米、垄距70-80厘米

### 1.2 种薯处理
- 选用脱毒种薯，单薯重30-50克为宜
- 播前15-20天出窖，在15-18°C散射光下催芽
- 芽长0.5-1厘米时切块，每块保留1-2个芽眼，重量25-30克
- 切刀用75%酒精消毒，切口蘸草木灰或甲基托布津粉剂

### 1.3 基肥施用（每亩）
- 腐熟有机肥2000-3000公斤
- 复合肥（15-15-15）40-50公斤
- 硫酸钾15-20公斤
- 硼砂1公斤
- 基肥沟施，与种薯间隔5厘米以上

## 二、播种技术

### 2.1 播种时期
- 锡林浩特地区：4月下旬至5月上旬
- 10厘米地温稳定在7-8°C以上
- 晚霜期前15-20天

### 2.2 播种规格
- 株距25-30厘米，行距70-80厘米
- 播深10-12厘米（沙壤土12厘米，粘土8-10厘米）
- 每亩保苗3500-4500株

## 三、各生长阶段管理

### 3.1 出苗期（播后35-49天）
- 及时查苗补苗，缺苗率超过10%需补种
- 出苗70%时进行第一次中耕培土，培土厚度5-8厘米
- 注意防治地下害虫（蛴螬、金针虫）

### 3.2 苗期（播后50-64天）
- 苗高15厘米时追施尿素10-15公斤/亩
- 第二次中耕培土，培土厚度8-10厘米
- 防治蚜虫：10%吡虫啉可湿性粉剂2000倍液喷雾

### 3.3 现蕾期（播后65-77天）
- 追施硫酸钾15-20公斤/亩 + 尿素5-8公斤/亩
- 第三次培土，形成高垄
- 预防晚疫病：72%霜脲·锰锌可湿性粉剂600倍液

### 3.4 开花/块茎形成期（播后78-91天）
- 保持土壤湿度60-70%，干旱时及时灌溉
- 叶面喷施0.3%磷酸二氢钾溶液
- 晚疫病防治：每7-10天喷药一次

### 3.5 块茎膨大期（播后92-112天）
- 水分管理最关键，保持均匀供水
- 禁止追施氮肥，防止徒长
- 继续防治晚疫病

### 3.6 成熟收获期（播后128天以后）
- 收获前7-10天杀秧，促进薯皮老化
- 选晴天收获，避免机械损伤
- 入窖前晾晒2-3天，剔除病薯烂薯

## 四、灌溉管理
- 全生育期需水量300-400毫米
- 现蕾至块茎膨大期为需水关键期
- 采用滴灌或喷灌，避免大水漫灌
- 收获前15天停止灌溉
