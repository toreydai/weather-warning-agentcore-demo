# 马铃薯病虫害防治手册

## 一、主要病害

### 1.1 晚疫病（最重要病害）
- **症状**：叶片出现水浸状暗绿色病斑，潮湿时叶背产生白色霉层
- **发病条件**：气温15-22°C，相对湿度>90%，连续阴雨
- **防治方案**：
  - 预防：现蕾期开始，72%霜脲·锰锌可湿性粉剂600倍液，每7-10天一次
  - 发病初期：68.75%氟吡菌胺·霜霉威悬浮剂600-800倍液
  - 严重时：50%烯酰吗啉可湿性粉剂1500倍液 + 80%代森锰锌600倍液交替使用
  - 安全间隔期：7-14天

### 1.2 早疫病
- **症状**：叶片出现同心轮纹状褐色病斑
- **发病条件**：气温25-30°C，植株衰弱时易发
- **防治方案**：
  - 75%百菌清可湿性粉剂500倍液
  - 10%苯醚甲环唑水分散粒剂1500倍液
  - 每10天喷施一次，连续2-3次

### 1.3 黑胫病
- **症状**：茎基部变黑腐烂，植株萎蔫
- **防治**：种薯消毒，72%农用链霉素4000倍液灌根

### 1.4 环腐病
- **症状**：维管束变褐，挤压茎切面有乳白色菌脓
- **防治**：严格种薯检疫，切刀消毒，发现病株立即拔除

## 二、主要虫害

### 2.1 蚜虫
- **危害**：刺吸汁液，传播病毒病
- **防治方案**：
  - 10%吡虫啉可湿性粉剂2000倍液喷雾
  - 3%啶虫脒乳油1500倍液
  - 黄板诱杀：每亩悬挂20-25块

### 2.2 蛴螬（金龟子幼虫）
- **危害**：咬食种薯和块茎
- **防治方案**：
  - 播种时沟施3%辛硫磷颗粒剂2-3公斤/亩
  - 5%毒死蜱颗粒剂3公斤/亩拌土撒施
  - 成虫期用频振式杀虫灯诱杀

### 2.3 金针虫
- **危害**：钻蛀块茎形成孔洞
- **防治**：同蛴螬，播前土壤处理

### 2.4 二十八星瓢虫
- **危害**：取食叶肉，留下网状叶脉
- **防治**：4.5%高效氯氰菊酯乳油1500倍液

## 三、农药使用安全

### 3.1 注意事项
- 严格按照推荐浓度配药，不可随意加大用量
- 喷药时间选择上午10点前或下午4点后
- 风力超过3级不宜喷药
- 收获前严格遵守安全间隔期
- 交替使用不同作用机理的农药，延缓抗药性

### 3.2 常用农药安全间隔期
| 农药 | 安全间隔期 |
|------|-----------|
| 霜脲·锰锌 | 14天 |
| 烯酰吗啉 | 7天 |
| 百菌清 | 7天 |
| 吡虫啉 | 7天 |
| 高效氯氰菊酯 | 7天 |
