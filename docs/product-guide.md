# 薯问 — 产品功能说明

> 更新时间：2026-05-15

薯问是一套面向马铃薯种植户、农艺师和审核员的智能田间管理系统，覆盖天气、预警、农事建议、知识库和 AI 问答。

---

## 1. 核心能力

### 生育进程追踪

- 顶部进度条展示 6 个主要生育阶段：播种-出苗 → 发棵 → 现蕾 → 开花结薯 → 块茎膨大 → 成熟收获
- 显示当前阶段名称、已过天数、距下一阶段天数
- 基于播种日期和播后天数（DAP）自动推算，实时更新

### 播种至今累计指标

- **有效积温（GDD）**：采用修正法（基温 7°C，上限 30°C），从播种日累计至今
- **累计降水**：从播种日累计至今
- 与去年同期对比（按播后天数 DAP 对齐），显示升高/降低趋势和具体差值
- 数据来源：Open-Meteo ERA5 历史再分析 + Open-Meteo 预报，每日 22:00 更新

### 气象适宜度评分

- **马铃薯气候适宜度**：基于近 14 天实况，温度/水分/昼夜温差加权（0-100 分）
- **植保适宜度**：基于未来 3 天预报，无雨 + 风小 + 温适中 = 高分
- **施肥适宜度**：基于未来 7 天预报，施肥后有小雨 = 高分
- **灌溉适宜度**：基于未来 7 天预报，降水缺口大 + 无雨预期 = 高分
- 水分阈值按生育阶段分别设定（块茎膨大期需水最多）
- v1 参数参考 FAO 马铃薯种植指南，待 QX/T 229-2014 标准采购后升级

### 天气与预报

- 查看每块地历史天气、未来天气预报（7 / 15 / 45 天切换）
- 展示气温、降水、风速、湿度、阵风、土温等指标
- 趋势图展示气温和降水变化，历史实况与预报无缝拼接
- 月份筛选限定在 45 天预报窗口内
- 与历史同期对比（气候参考值），辅助判断偏暖、偏冷、偏旱、偏湿

### 多年气象趋势对比图

三张独立图表，各自可选对比年份（2015-2025 任意年）：

| 图表 | 要素 | 对比 |
|---|---|---|
| 气温趋势 | 日最高/最低/平均气温、昼夜温差 | 当年 vs 选定年 vs 历史均值 |
| 降水趋势 | 日累计降水 | 当年（蓝柱）vs 选定年（绿柱）vs 历史均值（黄线） |
| 积温/累计降水对比 | 有效积温或累计降水（可切换） | 当年 vs 多选年份（最多 4 年），支持日/周/月聚合，"最大值"按钮显示各年峰值 |

数据来源：Open-Meteo ERA5 历史再分析（2015-2025）+ Open-Meteo 预报（当年）。

### 地块管理

- 添加、编辑、删除地块
- 记录经纬度（精确到小数点后两位）、面积、品种、播种日期
- 支持省/市/县/乡镇定位字段

### 每周田间管理报告

- 只展示当前周和下一周（共 2 条）
- 下一周报告标注"预测 · 本周末更新"
- 每周一 07:00 自动更新，下一周报告基于预报数据生成
- 支持快速规则生成和 AI 智能生成，支持人工编辑和审核
- 人工编辑过的建议不会被 AI 覆盖

### 预警体系

系统有三类预警：

| 类型 | 粒度 | 说明 |
|---|---|---|
| 灾害预警 | 地块 | 霜冻、暴雨、大风、高温等，按气象局标准阈值自动触发；仅检测未来 7 天，每日全量刷新 |
| 每日农事预警 | 县级 | 一县一天一条，结合 45 天预报、生育阶段和知识库生成 |
| 产区预警 | 产区 | 基于镇级气象数据和可配置阈值，每日 06:15 自动检测 |

### 产区视图

访问 `/zones/[id]` 查看产区概览：

- 聚合统计卡片（成员地块数、区域内降水均值等）
- 降水排名表（各成员镇排序）
- 成员明细表（各镇详情）
- 底图：天地图矢量图 + 中文注记
- 区域边界（OSM 行政边界优先，无数据时退回 8km buffer 圆）
- CircleMarker 散点（大小/颜色按降水分级）
- 7 天降水预报柱状图

### 智能问答

右下角聊天按钮支持自然语言提问，可询问天气、农事、预警、病虫害等。

### Public API（外部集成）

第三方系统可通过 OAuth 2.0 Client Credentials 授权访问 Public API：

- `POST /api/v1/oauth/token` — 获取 Bearer Token
- `GET /api/v1/public/weather/forecast` — 地块天气预报（1-45 天）
- `GET /api/v1/public/alerts/active` — 地块当日活跃预警
- `GET /api/v1/public/advice/daily` — 县级每日农事建议

API 规范（OpenAPI 3.1.0）：`infra/public-api-openapi.yaml`

## 5. 常见问题

| 问题 | 回答 |
|---|---|
| 预报最远看几天？ | 45 天，页面可切换 7 / 15 / 45 天 |
| 45 天都一样可靠吗？ | 不是。0-7 天高置信，8-15 天看趋势，16-45 天只做资源和作业窗口预判 |
| 有效积温怎么算的？ | 修正法：Tmin 按 7°C 封底，Tmax 按 30°C 封顶，取均温减基温 7°C，每日累加 |
| 去年同期怎么对齐的？ | 按播后天数（DAP）对齐，不是按日历日期，更符合农事实际 |
| 适宜度评分依据什么标准？ | v1 参考 FAO 马铃薯种植指南，后续升级到 QX/T 229-2014 气象行业标准 |
| 历史均值是哪年到哪年？ | 当前为气候参考值，来源待确认，后续会在 UI 上标注具体年份范围 |
| 每日县级预警和灾害预警有什么区别？ | 灾害预警按地块和阈值触发，监测未来 7 天每日刷新；每日县级预警每天固定生成一条农事指导 |
| AI 建议会覆盖手动编辑吗？ | 不会 |
| 手机上能用吗？ | 可以，页面和聊天面板均适配移动端 |

## 6. 当前限制

- HTTPS 暂未启用，当前通过 ALB HTTP 访问
- 历史均值（`historical_monthly`）数据来源待确认，UI 已标注"来源待确认"
- 适宜度评分为 v1 经验参数，待农艺师评审后调整
- 产区地图：旧口/柴湖镇无 OSM 行政边界，当前退回 8km buffer 圆；需 Lambda 中转或手动 GeoJSON
- 区域分布图（按要素预报等级的地图）待与Open-Meteo沟通栅格数据后实现
- Staging 环境暂未建立，当前通过 smoke + rollback 控制发布风险
