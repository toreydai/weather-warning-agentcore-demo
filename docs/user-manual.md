# 薯问 AgentCore — 用户手册

> 更新时间：2026-05-15

---

## 1. 登录

访问：

```text
http://weather-warning-agentcore-alb-54329175.us-east-1.elb.amazonaws.com
```

输入管理员分配的用户名和密码。首次登录或密码过期时，系统会要求修改密码。

密码要求：至少 8 位，包含大写字母、小写字母和数字，不能使用历史密码。

---

## 2. 首页与地块

首页展示所有地块卡片，包括名称、品种、面积、位置和坐标（精确到小数点后两位）。顶部导航还包含：**🗺️ 产区管理**（进入产区列表）和 **🔑 API 客户端**（管理员专用，OAuth 客户端管理）。

可操作：添加地块、编辑地块、删除地块（不可恢复）、进入地块详情。

添加地块时建议填写：名称、经纬度、省/市/县/乡镇、面积、品种、**播种日期**（影响生育阶段判断和农事建议）。可选字段（乡镇、面积、品种、播种日期、详细地址）留空即可，系统不会强制要求填写；若提交后页面提示具体字段错误，按提示修正即可。

---

## 3. 地块详情

进入地块后，页面从上到下依次展示：

### 3.1 生育进程进度条

显示当前所处的生育阶段（播种-出苗 / 发棵 / 现蕾 / 开花结薯 / 块茎膨大 / 成熟收获），以及：

- 已过天数（播后天数 DAP）
- 距下一阶段还有几天
- 当前细分阶段名称

进度条颜色：已过阶段（深绿）→ 当前阶段（中绿）→ 未到阶段（浅绿）。

### 3.2 播种至今累计指标

两张卡片并排：

| 卡片 | 内容 |
|---|---|
| 有效积温 | 播种日至今累计 GDD（°C·d），与去年同期对比，显示升高/降低趋势 |
| 累计降水 | 播种日至今累计降水（mm），与去年同期对比 |

去年同期按播后天数（DAP）对齐，不是按日历日期。

### 3.3 统计卡片

| 卡片 | 内容 |
|---|---|
| 季平均气温 | 当前筛选月份的日均温均值 |
| 本月累计降水 | 当月日降水累加 |
| 今日天气 | 最高/最低温和天气描述 |
| 未来 7 天极端天气预警 | 预警次数和等级分布 |

### 3.4 未来天气预报

可切换 7 / 15 / 45 天：

- 0-7 天：高置信，可用于具体作业安排
- 8-15 天：中置信，适合看趋势
- 16-45 天：低置信，只用于资源、人手和农资准备

### 3.5 气象适宜度评分

4 项进度条评分（0-100）：

| 评分 | 说明 |
|---|---|
| 马铃薯气候适宜度 | 基于近 14 天实况，综合温度/水分/昼夜温差 |
| 植保适宜度 | 基于未来 3 天预报，无雨 + 风小 = 高分 |
| 施肥适宜度 | 基于未来 7 天预报，施肥后有小雨 = 高分 |
| 灌溉适宜度 | 基于未来 7 天预报，降水缺口大 = 高分 |

等级：适宜（80+）/ 较适宜（60-79）/ 一般（40-59）/ 不适宜（<40）。

### 3.6 月份筛选 + 每周田间管理报告 / 极端天气预警

月份筛选器限定在未来 45 天窗口内。

**每周田间管理报告**只展示当前周和下一周：

- 当前周：基于实况 + 预报生成
- 下一周：标注"预测 · 本周末更新"，每周一自动更新

点击报告卡片展开查看每日天气、综合评估、施肥/植保/灌溉/田间管理建议。

**极端天气预警**展示未来 7 天的霜冻、暴雨、大风、高温等预警，每日 06:10 更新。

### 3.7 气温趋势图

折线图展示日最高/最低/平均气温和昼夜温差，右上角可选对比年份（2015-2025 任意年），叠加历史均值参考线。

### 3.8 降水趋势图

柱状图展示日降水量，右上角可选对比年份，叠加历史同期均值折线。

### 3.9 积温 / 累计降水对比图

- 顶部切换：📈 有效积温 / 🌧️ 累计降水
- 右上角多选年份（最多 4 年同时对比）
- 日/周/月聚合切换
- "最大值"按钮：显示各年的峰值水平线，方便横向比较；点"恢复"回到原始曲线

### 3.10 与历史同期比较

各月气温/降水/风速/湿度与气候参考值对比，差值正负一目了然。

## 4. 智能助手

点击右下角聊天按钮打开 AI 助手。

**两种模式：**

- **全局问答**（从首页打开）：不绑定具体地块
- **地块问答**（从地块详情页打开）：AI 自动带入该地块数据，回答更精准

**示例：**

| 类型 | 示例 | 速度 |
|---|---|---|
| 问候 | "你好" | ~1 秒 |
| 预警 | "有没有霜冻风险" | ~1 秒 |
| 综合 | "这周综合分析一下" | ~2 秒 |
| 天气 | "最近天气怎么样" | ~5 秒 |
| 农事 | "这周要不要追肥" | ~5 秒 |
| 病虫害 | "晚疫病怎么防治" | ~8 秒 |

## 5. 农事建议

周报中有两种生成方式：快速生成（规则，秒级）和 AI 智能生成（查询天气和知识库）。

建议可编辑和审核。人工编辑过的建议不会被 AI 自动覆盖。

标签含义：🤖 Agent 生成 / ✨ AI 生成 / ⚙️ 系统生成 / ✏️ 人工编辑 / ✅ 已审核。

## 6. 预警

### 地块级灾害预警

自动监测未来 7 天预报，触发类型：霜冻、暴雨、大风/阵风、高温、寒潮、暴雪、低温冷害、台风。

等级：黄色（关注）/ 橙色（警告）/ 红色（严重）。每天 06:10 自动刷新。

### 县级每日农事预警

一县一天一条，由系统结合县内所有地块预报、生育阶段、知识库生成草稿，审核员发布后显示在地块详情页。

### 产区预警

进入产区视图（`/zones/[id]`）可查看该产区的气象预警，基于各成员镇气象数据和可配置阈值，每日 06:15 自动检测。

---

## 7. 数据导出

地块详情页支持导出气象 CSV（每日气温/降水/风速等）和预警 CSV（预警类型/等级/触发日期）。

## 8. 常见问题

| 问题 | 回答 |
|---|---|
| 有效积温怎么算的？ | 修正法：Tmin 按 7°C 封底，Tmax 按 30°C 封顶，取均温减 7°C，每日累加 |
| 去年同期怎么对齐？ | 按播后天数（DAP）对齐，不是按日历日期 |
| 适宜度评分依据什么？ | v1 参考 FAO 标准，后续升级到 QX/T 229-2014 |
| 积温图的"最大值"是什么意思？ | 显示该年从播种日到当前的积温峰值，方便多年横向比较 |
| 为什么只看到 2 条周报？ | 只展示当前周和下一周，下一周为预测，每周一更新 |
| 预报为什么有 45 天？ | 用于提前安排农资、人手和作业窗口，但越远越不确定 |
| 聊天记录会丢吗？ | 本地浏览器保存历史，刷新可恢复；服务端 Memory 保存上下文 |
| 手机上能用吗？ | 可以 |
